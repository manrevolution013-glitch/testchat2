# Favicon Generation Guide

For sites other than cuckchat (which uses `/favicon_io`), generate first-letter favicons:

1. Go to https://favicon.io/favicon-generator/
2. Enter the first letter (e.g., "L" for lesbianchat)
3. Choose a background color (matching site theme)
4. Download the generated favicon package
5. Extract to `/public/favicons/{sitename}/` folder

Required files:
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- android-chrome-192x192.png
- android-chrome-512x512.png
